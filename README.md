# Toric Ideals of Graphs – Computational Code

This repository contains the Python code used for the computational
part of my project on toric ideals of graphs.

The tasks which the scripts perform include (but are not limited to) the following:

- enumeration of non-isomorphic graphs
- detection of induced subgraphs
- generation of Macaulay2 input files
- computation of primitive even closed walks

The generated Macaulay2 scripts are used to compute toric ideals,
minimal generators, and Castelnuovo–Mumford regularity.

The repository accompanies the written report.
